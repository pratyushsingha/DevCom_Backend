
# DevCom Backend

## Run Locally

Clone the project

```bash
  git clone https://github.com/pratyushsingha/DevCom_Backend
```

```bash
  cd DevCom_Backend
```

Install dependencies

```bash
  bun install
```

Start the server

```bash
  bun run dev
```
server is  running at `http://localhost:8080/`
